# Car Issues Web App
#### By John Tzanidakis
A Crowd-sourced web application for tracking issues with cars. Users can submit issues they have had with their cars, upvote/downvote other issues listed.

### Technologies Used:
* Node.js, Express.js
* AngularJS
* MongoDB
* HTML5, CSS3, Bootstrap

Web app is live at [carissues.herokuapp.com](https://carissues.herokuapp.com)
